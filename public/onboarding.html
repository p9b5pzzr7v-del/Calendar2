✦ NOTION API KEY
notion.so/my-integrations에서 인테그레이션을 만들고 Internal Integration Secret을 입력해 주세요.
API 키를 입력해 주세요
✦ DATABASE ID
노션 DB 페이지 URL에서 ID를 복사해 주세요.
notion.so/워크스페이스/여기가 DB ID?v=...
하이픈(-) 포함 또는 제외 모두 가능해요
DB ID를 입력해 주세요
💡 DB를 인테그레이션에 연결했는지 확인해 주세요 — DB 페이지 우상단 ··· → 연결 추가
✦ DB 속성 이름
노션 DB의 실제 속성 이름을 그대로 입력해 주세요. 비워두면 기본값을 사용해요.
진행중 상태인 항목은 중요(핑크)로 표시돼요
✦ 설정 완료
Vercel 환경변수 설정
    Vercel → 프로젝트 → Settings → Environment Variables에 위 값들을 추가해 주세요.
    ✦ 배포 가이드
    1
    vercel.com에서 이 레포를 fork하거나 새로 배포해 주세요.
    2
    Settings → Environment Variables에서 위 환경변수 3개를 추가해요.
    3
    Redeploy 후 노션에서 /embed → 배포된 URL 붙여넣기.
    4
    끝! 캘린더가 내 DB랑 연동돼요 🩵
